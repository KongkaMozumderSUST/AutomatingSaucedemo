# AutomatingSaucedemo
Automating saucedemo, A user logs in with valid credentials, adds one product to the cart, verifies the product name in the cart, and logs out.

### Clone the repository 

    git clone https://github.com/KongkaMozumderSUST/AutomatingSaucedemo.git

### Install Playwright

    npm init -y
    npm install @playwright/test
    npx playwright install

### Run test
    npx playwright test
